# Simple-Egyptian-Medical-AI-Chatbot
Simple Egyptian Medical AI Chatbot built with Streamlit, Large Language Models (LLMs), and Retrieval-Augmented Generation (RAG). The chatbot provides accurate, context-aware responses to health-related questions by retrieving relevant medical information from a knowledge base and generating user-friendly answers through an interactive web interface
.Features
  Medical question answering
  Context-aware responses using RAG
  Interactive web interface with Streamlit
  Integration with a medical knowledge base
  Easy to customize and extend
.Technologies Used
  Python
  Streamlit
  Large Language Models (LLMs)
  Retrieval-Augmented Generation (RAG)
  Vector Database
